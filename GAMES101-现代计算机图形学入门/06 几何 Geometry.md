### 1.2 表达几何的方式
#### 1.2.1 隐式几何 "Implicit" Representations of Geometry
**隐式几何的特点：**
+ 给出点满足某些特定关系
+ 不直观，很难判断面上的点在哪
+ 容易判断点在内部还是外部
+ 通常将满足的关系化为 $f(x, y, z) = 0$ 的形式

##### 1.2.1.3 距离函数 Distance Functions
**距离函数**描述的是：空间中的任意一点到想要表述的几何形体上的任意一点的最小距离(外正或内负，空间中任何一点都定义出来一个值)
这种方法是通过 **融合(Blend)** 距离函数的方式来描述新的物体，得到融合后的距离函数后，可以令距离函数为0时的位置为表面

##### 1.2.1.4 水平集 Level Set Methods
用一个显式数学方程来精确表示复杂几何形状通常非常困难
替代方法是：存储一个网格上的数值来近似表示这个函数，然后通过插值找出为 0 的表
![[Pasted image 20260607154236.png]]
可以提供对形状的更明确的控制(如纹理)
在地理上有类似概念——等高线
- 在CT、核磁共振等得到的医疗数据(如密度)中，可以找到使密度函数等于某个密度的集合，通过水平集的方法找到表面 Level sets encode, e.g., constant tissue density
- 水滴在落下四溅的过程中也可以通过定义距离函数或水平集的方式表述水滴和水滴融合在一起后的表面

##### 1.2.1.5 分形 Fractals
+ 表现出自相似性
- 经常被用来描述一些自然现象
- 难以控制形状
![[Pasted image 20260607154539.png]]

##### 1.2.1.6 隐式几何的优缺点 Implicit Representations - Pros & Cons
+ 优点
	+ 简洁的描述(如一个函数)
	+ 某些查询容易(物体内部，到表面的距离)
	+ 利于求光线到表面的交集
	+ 对于简单的形状，描述准确/没有采样错误
	+ 容易处理拓扑结构的变化(如流体)
+ 缺点
	+ 难以创建复杂的形状

#### 1.2.2 显式几何 “Explicit” Representations of Geometry
**显式几何的特点**
+ 所有的点都直接或通过参数映射给出
+ 容易判断面上的点在哪
+ 难以判断点在内部还是外部
+ 除了三角形面，通过参数映射定义的方法也很常见

##### 1.2.2.1 点云 Point Cloud
+ 最简单的表示方法：点列表(x,y,z)
+ 轻松表示任何类型的几何体
+ 适用于足够大型的数据集 (>>1 点/像素)
+ 通常转换为多边形网格
+ 难以使用欠采样的数据进行绘制

##### 1.2.2.2 多边形面 Plygon Mesh
+ 存储顶点和多边形(通常是三角形或四边形)
+ 更容易进行处理/模拟，自适应采样
+ 更复杂的数据结构
+ 也许是图形中最常见的表示

### 2.2 贝塞尔曲线 Bézier Curves
贝塞尔曲线是需要一系列的控制点去定义的一个曲线，这个曲线满足一些性质，比如下图的要沿着切线方向
![[Pasted image 20260607190205.png]]

#### 2.2.1 Bézier Curves – de Casteljau Algorithm
**二次贝塞尔曲线的计算：**
1. 定义三个顶点（二次贝塞尔曲线）
2. 根据 t 值在两条边上分别插值出两个点
3. 在连接两个点得到的线段上继续应用 t 值插值
4. 在 $[0, 1]$ 上遍历 t ，最终插值得到的点的移动路径即贝塞尔曲线
![[GIF 2022-2-4 6-02-46.gif]]

三次贝塞尔曲线同理

#### 2.2.2 贝塞尔曲线的代数表示 Bézier Curve – Algebraic Formula
de Casteljau 算法给出了这样一个代数金字塔
![[Pasted image 20260607191020.png]]

三次贝塞尔曲线的代数表示如下，写出后发现各项系数正好是二项式平方 $((1 - t) + t)^2$ 展开的结果
![[Pasted image 20260607193137.png]]

推广到 n 阶，展开项系数是一个描述二项分布的多项式（可以用伯恩斯坦公式来表达）

当 $n = 3$ 时，我们可以用来表示三维空间中的贝塞尔曲线

b^3(t) = b_0(1 - t)^3 + b_13t(1 - t)^2 + b_23t^2(1 - t) + b_3t^3
$$

#### 2.2.3 贝塞尔曲线的性质
+ 对于三次贝塞尔曲线，插值的端点：$b(0) = b_0, b(1) = b_3$
+ 对于三次贝塞尔曲线，与端点相切的切线表示的系数为3：$b'(0) = 3(b_1 - b_0), b'(1) = 3(b_3 - b_2)$ 
+ 在仿射变换时，只需要对顶点做仿射变换，就能得到这个贝塞尔曲线在仿射变换下的结果（只限于时仿射变换，投影变换时不成立）
+ 贝塞尔曲线在控制点形成的凸包(能够包围给定点的最小凸多边形)内 Curve is within convex hull of control points
![[Pasted image 20260607194235.png|450]]

#### 2.2.4 分段贝塞尔曲线 Piecewise Bézier Curves
当控制点较多时，贝塞尔曲线不直观且不易控制
![[Pasted image 20260607194342.png]]

为了便于控制曲线，可以定义多段贝塞尔曲线然后连起来，普遍习惯每四个控制点定义一段
![[Pasted image 20260607194647.png]]

#### 2.2.5 贝塞尔曲线的连续性 Piecewise Bézier Curve – Continuity
我们用连续性来定义分段贝塞尔曲线的光滑程度
+ $C^0$ 连续：只需要点相同 $a_n = b_0$![[Pasted image 20260607194918.png]]
+ $C^1$ 连续：点相同、切线方向以及大小都要相同 $a_n = b_0 = \frac{1}{2}(a_{n-1} + b_1))$ ![[Pasted image 20260607195128.png]]
一般工业上需要到达 $C^2$ ，即二阶导数也相同

## 4 几何处理 Geometry Processing
### 4.1 网格细分 Mesh Subdivision
通常的三角形网格细分规则：
1. 创造更多的三角形（顶点）
2. 调整顶点的位置
![[Pasted image 20260626100005.png]]

#### 4.1.1 Loop Subdivision
1. 连接各边中点，一个三角形会被分为 4 个三角形![[Pasted image 20260626100027.png]]
2. 通过权重更新新顶点位置，新旧顶点更新规则不同
   + 新顶点位置更新为：$3 / 8 * (A + B) + 1 / 8 * (C + D)$ ![[Pasted image 20260626100425.png]]
   + 旧顶点位置更新为：$(1 - n * u) * original_position + u * \text(neighbor\_position\_sum)$ ![[Pasted image 20260626100844.png]]
Loop 细分的结果：

#### 4.1.2 Catmull-Clark Subdivision (General Mesh)
Catmull-Clark Subdivision 可以适用于四边形等常见的网格， Loop Subdivision 主要适用于三角形
1. 定义度不为 4 的顶点为奇异点，如下图紫色点 ![[Pasted image 20260626101431.png]]
2. 在一次细分过程中，取边上的中点和面中心上的一个点，把这些点连起来 ![[Pasted image 20260626101652.png]]
3. 一次细分结束后，会引入了两个度为 3 的奇异点，一共有 4 个奇异点；也就是说一次细分会将非四边形面都变为四边形，并对应地引入一个新的奇异点 ![[Pasted image 20260626102229.png]]
4. 继续细分，奇异点数目已经不会改变了

#### 4.2.1 边塌缩 Collapsing An Edge
这里将中间的那条边塌缩了，拥有这条边的两个平面会被删除
![[Pasted image 20260626102927.png]]
具体实现：
+ 定义⼆次误差度量 (Quadric Error Metrics)，这里的二次误差即新顶点到原来附近所有相关面的平面距离平方累加，我们需要这个误差最小 ![[Pasted image 20260626103251.png]]
+ 简单假设每一条边如果坍缩会导致多大的二次度量误差，然后从二次度量误差最小的开始坍缩
+ 但是坍缩一条边后会改变其他的边的位置，其他边的二次度量误差也必须重新算，所以我们需要每次坍缩完二次度量误差最小的边后，动态的更新其他受影响的边，需要使用到的数据结构是优先队列/堆
+ 通过对局部进行最优解，试图找到全局的最优解，这是一个典型的贪心算法
